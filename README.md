# web_scraper
Web scraper, works with Image, Doc, Video

### Dependencies
 - python 3.6.0
 - click 6.7.0
 - requests 2.18.4
 - Pillow 4.3.0
 - selenium 3.7.0

### Usage

```
python3.6 scrape.py --url=https://unsplash.com --selector='#gridMulti img' --file_type=image --folder_output='./.images'
```
